Pipes are used to format the data before displaying in the View. Pipe is used by using |. This symbol is called a Pipe Operator.

Types of pre-defined or built-in pipes in Angular 6.

1.Lowercase
2.Uppercase
3.Titlecase
4.Slice
5.Json Pipe
6.Number Pipe
7.Percent Pipe
8.Currency Pipe
9.Date Pipe

1.Example Lowercase

{{lower|lowercase}}

2.Example UpperCase

{{upper|lowercase}}

3.Example Titlecase

Titlecase converts the first letter of a word in the uppercase.

{{name|titlecase}}

4.Example Slice

{{name1|slice:1:3}}

{{months | slice:2:6}}

JSON PIPE

{{ jsonval | json }}

6.Example Number Pipe

{{8.7844|number:'1.2-2'}}

{{ 454.78787814 | number: '3.4-4' }} // 3 is for main integer, 4 -4 are for integers to be displayed.

7.Example Percent Pipe

{{0.23|percent}}

8.Example Currency Pipe

{{600|currency}}

{{0.23|currency:'GBP'}}

{{506|currency:'INR'}}

9.Example Currency Pipe

{{ birthday | date }}

Note - Parameterizing a pipe
We can also pass a parameter to the pipe. You can write the HTML code like this to pass the parameter.

{{ birthday | date:"dd/MM/yyyy" }}

Chaining pipes

We can chain pipes together in and create useful combinations. We can use the uppercase or lowercase pipe in our example.

{{ birthday | date | uppercase }}
